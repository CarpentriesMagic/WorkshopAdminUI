```

chmod 600 ~/.ssh
ssh-keygen
```

## For Reference
### Basic Bash Commands

- `ls` list: Lists the files in your current directory
- `ls -a` list all: Lists all files, including hidden ones
- `mkdir <dirname>` make directory: Makes a new directory
- `rm <filename>` remove: Remove a file
- `rm -rf <dirname>`: remove recursively: Removes a directory and all files and directories below it (recursively)
- `cd <dirname>` change directory:  Change to specified directory
- `mv` move: Move a file to a different location or to a different name
